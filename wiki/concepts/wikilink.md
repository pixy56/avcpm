# Wikilink

`\[\[Page Name\]\]` syntax for bidirectional linking between markdown pages.
Used in Obsidian and this wiki for graph navigation.

## Syntax
- `\[\[Page Name\]\]` — link to a page by its title or filename
- `\[\[path/to/Page\]\]` — link with directory path
- `\[\[Page Name|Display Text\]\]` — link with custom display text

## Purpose
- Creates explicit, traversable relationships between knowledge nodes
- Enables backlink discovery (which pages link to this one?)
- Powers graph visualization in Obsidian and other tools
