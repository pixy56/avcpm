# Letta

Tiered memory architecture for stateful AI agents. ~22K GitHub stars.

## Origin
Born from MemGPT research at UC Berkeley. $10M seed funding.

## Architecture
- **Core memory** — Pinned to context window
- **Archival memory** — Evicted but retrievable
- **Recall memory** — Recently accessed
