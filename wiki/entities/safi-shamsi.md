# Safi Shamsi

## Role

London-based AI engineer. Creator of **Graphify**, an open-source (MIT) AI coding-assistant skill that converts any folder of files into a queryable knowledge graph.

## Key Contribution

Graphify: launched April 2026, ~36k GitHub stars. Provides a 7-stage pipeline for auto-extracting knowledge graphs from code, docs, PDFs, images, video, and audio. First-class OpenClaw integration.
