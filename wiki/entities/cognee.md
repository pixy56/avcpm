# Cognee

Graph-vector hybrid knowledge system with 14 retrieval modes. $7.5M seed.

## Pipeline
1. **Ingest** — Add documents
2. **Cognify** — Extract entities, relationships, build graph
3. **Search** — 14 retrieval modes (vector, graph, hybrid)

## Features
- Six-stage knowledge graph pipeline
- `memify` for self-improvement
- Default stores: LanceDB (vector), Kuzu (graph)
