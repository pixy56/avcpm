- File-based storage (plain `.md` files)
- YAML frontmatter for structured metadata
- Wikilinks (`\[\[Wikilink\]\]`) for graph navigation
- Massive plugin ecosystem (MCP servers, REST API, CLI)

- Letta, Agent Client Protocol, various agent plugins
